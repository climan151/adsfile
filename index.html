Microsoft Azure AI Fundamentals

Course Description

This beginner-level course introduces core concepts of artificial intelligence and how they're implemented using Microsoft Azure. It covers machine learning, computer vision, natural language processing, and conversational AI, with real-world use cases. No coding or AI experience is required—just curiosity and a desire to understand how AI works in the cloud.
